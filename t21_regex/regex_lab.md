# Regular expressions exercises

## Initial exercise

Use regex101 site to do the following:

Hints:
- https://www.postgresql.org/docs/current/functions-matching.html
- Table 9.21. Regular Expression Class-Shorthand Escapes

1. Find approx 10 email addresses
2. Construct regex for e-mail address that will match them.
3. Make sure that it does *not* match the following:
	- user@domain-com
	- user@computer
	- user@user@computer.com
4. Try out your regex support in Python.
