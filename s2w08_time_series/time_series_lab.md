title: Time series lab
---

1.  Download the [CSV version of the
    CovidStatisticsProfileHPSCIrelandOpenData
    set](https://data.gov.ie/dataset/covidstatisticsprofilehpscirelandopendata1)
    from <https://data.gov.ie/>.

2.  Create a table to hold all columns of the dataset. Apply appropriate
    constraints, e.g. `NOT NULL`. You can exclude duplicated columns.

3.  Repeat the analysis performed in the lecture notes. You can make
    changes as you wish (e.g. consider using Python to directly load the
    DB and/or create the columns rather than relying on CSV).

4.  Expand your analysis to include at least two additional columns.

5.  Identify at least one time series analysis method (that you have
    done in Python) that you can implement using the Window Function
    methods demonstrated. Implement this analysis and confirm that it
    agrees with your Python implementation.
